# Sviluppo-di-una-pagina-web-per-il-download-dei-report-di-sostenibilit-per-azienda-settore-primario
I report di sostenibilità sono documenti pubblicati dalle aziende per riferire sulle loro prestazioni ambientali, sociali ed economiche. Questi report sono essenziali per dimostrare l'impegno dell'azienda verso pratiche di sostenibilità e trasparenza
